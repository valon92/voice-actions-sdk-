## How Commands Work

1. **SDK listens** for voice input
2. **Matches transcript** to command phrases
3. **Calls `onCommand` callback** with command object
4. **Platform implements** the specific logic for each command

## Example Implementation

```javascript
const sdk = new VoiceActionsSDK({
    apiKey: 'your-api-key',
    platform: 'instagram',
    locale: 'en-US',
    onCommand: (command) => {
        // Handle command based on action
        switch (command.action) {
            case 'like':
                // Your Instagram like logic
                likeCurrentPost();
                break;
            case 'create-story':
                // Your Instagram story creation logic
                openStoryCreator();
                break;
            case 'go-live':
                // Your Instagram live stream logic
                startLiveStream();
                break;
            // ... handle all other commands
        }
    }
});
```

## Command Object Structure

Each command object contains:
```javascript
{
    id: 'like',                    // Unique command ID
    action: 'like',                // Action identifier
    phrases: ['like', 'heart'],    // Array of phrases that trigger this command
    description: 'Like post/content', // Human-readable description
    category: 'interaction'         // Command category
}
```

## Notes

- All commands are **universal** and work for any platform
- Platforms implement their own logic in the `onCommand` callback
- SDK only provides command detection and matching
- Commands are organized by category for easy reference
- Each command has multiple phrase variations for better recognition
